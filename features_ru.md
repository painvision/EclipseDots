# Функции
- Конфиги для fish, fastfetch и kitty 
> отдельная команда 'cavacolor' для cava в градиенте
- **Цветовые схемы на основе обоев** для всего
- **Функция обзора** для лучшего управления окнами и рабочими столами
- Утилита скриншотов с плавными **анимациями**
- **Панели окон** как в классических рабочих окружениях
- Приближение экрана, красивый курсор, игровой режим
- Горячие клавиши для контроля мышки с помощью клавиатуры
- Некоторые жесты для ноутбуков

А также все функции [Noctalia Shell](https://github.com/noctalia-dev/noctalia-shell):

- **Лаунчер приложений**, виджеты для Wi-Fi, Bluetooth, контроль громкости/яркости, календарь, панель уведомлений
- Приложение настроек
- Классное **меню "Пуск"** с **аудиовизуализатором** и **эффектами погоды**
- Калькулятор, **выбор эмодзи**, **буфер обмена** 
- Управление обоями, **вы можете искать и применять обои напрямую с панели**
- Менеджер профилей питания для ноутбуков
- Запись экрана
- Легко кастомизируемая паель
- Блокировка экрана
- Интеграция с **Walogram** для динамичных **Telegram тем** на основе обоев
