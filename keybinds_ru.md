# Базовое управление

`⌘ Super + ENTER ` - Терминал (kitty)

`⌘ Super + Q ` - Закрыть приложение

`⌘ Super + T ` - Переключить плавающие окна

`⌘ Super + R ` - Лаунчер приложений

`⌘ Super + (1-9) ` - Фокус на рабочий стол 1-9) 

`⌘ Super + SHIFT + (1-9) ` - Переместить активное окно в рабочий стол (1-9)

`⌘ Super + SHIFT + (стрелочки) ` - Изменить размер активного окна

`⌘ Super + CTRL + (стрелочки) ` - Двигать активное окно (в плавающем состоянии)

`⌘ Super + (стрелочки)` - Переместить фокус

`⌘ Super + SHIFT + R` - Перезагрузить Hyprland

`⌘ Super + K` - Поменять плитки окон местами

`⌘ Super + J` - Разделить плитки окон горизонтально

`⌘ Super + ЛКМ` - Двигать окно

`⌘ Super + ПКМ` - Менять размер окна

`⌘ Super + F` или `F11` - Полноэкранный режим

`PrtSc` - Полный скриншот

`⌘ Super + N ` - Настройки

# Продвинутое управление

`⌘ Super + P ` - Псевдо режим

`⌘ Super + H ` - Перейти в специальный рабочий стол

`⌘ Super + SHIFT + H ` - Переместить окно в специальный рабочий стол

`⌘ Super + D` - Спрятать панель

`⌘ Super + End` or `CTRL + SHIFT + ESC` - Диспетчер задач (btop)

`⌘ Super + I` - Приблизить экран

`⌘ Super + O` - Отдалить экран

`⌘ Super + SHIFT + (колесико мыши)` - Зум экрана

`⌘ Super + (колесико мыши)` - Скролл существующих рабочих столов

`⌘ Super + CTRL + (колесико мыши)` -  Скролл всех рабочих столов

`⌘ Super + (боковые кнопки мыши)` - Переключение рабочих столов

# Полезные инструменты

`⌘ Super + Tab` - Менеджер рабочих столов и окон

`⌘ Super + C ` - Калькулятор

`⌘ Super + SHIFT + C ` - Запись экрана

`⌘ Super + M ` - Меню "Пуск"

`⌘ Super + V` - Буфер обмена

`⌘ Super + L` - Блокировка экрана

`⌘ Super + SHIFT + L` - Выбор действий

`⌘ Super + PrtSc` - Красивый скриншот

`⌘ Super + ALT + G` - Игровой режим

`⌘ Super + Period "."` - Эмодзи

# Режимы питания для ноутбуков

`⌘ Super + CTRL + S` - Режим экономии энергии

`⌘ Super + CTRL + B` - Режим баланса

`⌘ Super + CTRL + P` - Режим производительности

# Приложения

`⌘ Super + E ` - Файловый менеджер (nemo)

`⌘ Super + B` - Firefox

# Управление уведомлениями

`⌘ Super + SHIFT + N ` - Режим 'Не беспокоить'

`⌘ Super + CTRL + N ` - Очистить все уведомления

# Управление обоями

`⌘ Super + CTRL + w` - Выбор обоев

`⌘ Super + SHIFT + W` - Случайные обои

# Контроль мышки клавиатурой

`ALT + (стрелочки)` - Контроль курсора

`ALT + CTRL (стрелочки)` - Точный контроль курсора

`ALT + BACKSPACE` - Клик левой клавишей мыши

`ALT + CTRL + BACKSPACE` - Клик правой клавишей мыши
